# Yarn

## CLI

```sh
# 显示当前配置
$ yarn config list
# 查看当前使用的镜像源
$ yarn config get registry
# 临时修改
$ yarn save 软件名 --registry https://registry.npm.taobao.org/
# 全局修改
$ yarn config set registry https://registry.npm.taobao.org/
```
