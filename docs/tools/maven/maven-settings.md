# Maven Settings

## Mirrors

```xml
<mirrors>
  <mirror>
    <id>huaweicloud</id>
    <name>huawei maven mirror</name>
    <mirrorOf>central</mirrorOf>
    <url>https://mirrors.huaweicloud.com/repository/maven</url>
  </mirror>
  <mirror>
    <id>aliyun</id>
    <name>aliyun maven mirror</name>
    <mirrorOf>central</mirrorOf>
    <url>https://maven.aliyun.com/repository/central</url>
  </mirror>
</mirrors>
```
