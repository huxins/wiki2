# Core Specifications

HTTP/1.1 由以下规范定义：

  • RFC7230: HTTP Message Syntax and Routing (opens new window)
  • RFC7231: HTTP Semantics and Content (opens new window)
  • RFC7232: HTTP Conditional Requests (opens new window)
  • RFC7233: HTTP Range Requests (opens new window)
  • RFC7234: HTTP Caching (opens new window)
  • RFC7235: HTTP Authentication (opens new window)

这些 RFC 共同淘汰了之前定义 HTTP 的所有 RFC，包括 RFC1945 (opens new window)、RFC2068 (opens new window)、RFC2616 (opens new window) 和 RFC2617 (opens new window)。
